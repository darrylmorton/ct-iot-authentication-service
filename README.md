# ct-iot-things-report-request-service

## Description

The `authentication-service` verifies JWTs, and creates them for login.

[Diagrams](./docs/DIAGRAMS.md)

## Requirements

python v3.11.9+  
poetry v1.7.1

## Build

```
make build
```

## Run

```
make server-start
```

### Test

```
docker-compose up -d
```

### Run Tests

```
make test
```
